# StockTill

库存管理与收银系统 Inventory & Checkout System

## 安装

1. 下载并安装 SQL Server 2022 Express：https://www.microsoft.com/zh-cn/sql-server/sql-server-downloads
2. 下载并安装 StockTill：https://github.com/St7530/StockTill/releases
   请确保您有 .NET 8.0 运行环境。
